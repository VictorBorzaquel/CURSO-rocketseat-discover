# Global Value

|color|Description|
|-|-|
|inherit|pega a cor do parent|
|initial|volta a cor original|
|currentcolor|pega a cor atual|
|unset|não definir nenhuma cor ( acaba pegando a cor do contexto )|

# Hexadecimal

Vai de: 0 - f

## Menos controle da cor:

#f09

#f095

- red = f
- green = 0
- blue = 9
- opacity = 5 ( opicional )

<!-- red, green, blue, opacity -->

## Mais controle da cor:

#ff0099

#ff009950

- red = ff
- green = 00
- blue = 99
- opacity = 50 ( opicional )
<!-- red, green, blue, opacity -->

# RGB

Vai de 0 - 255

rgb( 255, 12, 120 )

rgba( 255, 12, 120, .5 )

- red = 255
- green = 12
- blue = 120
- opacity = 50% ( opicional )

# HSL

![](./hsl.ppm)

hsl( 180, 30%, 50% )

hsla( 180, 30%, 50%, 70% )

- Hue = 180º
- Saturation = 30%
- Light = 50%
- opacity = 70% ( opicional )
